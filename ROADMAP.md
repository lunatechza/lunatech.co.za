# Lunatech Roadmap: Vision 2026

This roadmap outlines the evolution of the Lunatech (Pty) Ltd. digital presence as we transition into a fully modernized, AI-integrated software powerhouse.

## Phase 1: AI Branding & Port (Completed Q2 2024)
- [x] Decommission legacy site and port to GitHub Pages.
- [x] Update core narrative to "Vision 2026" focusing on AI & Automation.
- [x] Standardize global branding to "Design. CODE. AI."
- [x] Refresh team and project portfolios.
- [x] Implement initial 2026 "AI Insights" UI accents.

## Phase 2: Modern Tech Stack Migration (Target: Q3-Q4 2024)
- [ ] **Framework Upgrade:** Migrate from Bootstrap 3 / jQuery to a modern React-based framework (e.g., Next.js).
- [ ] **Styling:** Transition to Tailwind CSS for more flexible and high-performance UI components.
- [ ] **Iconography:** Upgrade to SVG-based icons (Heroicons/Lucide) to enable modern AI metaphors (Robots, Neural Nets).
- [ ] **Performance:** Achieve 95+ Lighthouse scores for SEO and accessibility.

## Phase 3: Factual Data & Real-Time AI (Target: 2025)
- [ ] **Live AI Integration:** Embed a proprietary Lunatech AI Assistant trained on company whitepapers and projects.
- [ ] **Real-Time Portfolio:** Connect Portfolio items to live GitHub/Status metrics for "factual" performance data.
- [ ] **Dynamic Roadmaps:** Automate roadmap updates based on internal project management milestones.

## Phase 4: Full 2026 Launch (Target: Jan 2026)
- [ ] **Immersive UI:** Deploy a "Spatial Design" or "Bento Box" aesthetic reflecting mid-2020s design trends.
- [ ] **Autonomous Content:** Transition blog/news sections to autonomous curation by LunaAI agents.
- [ ] **Factual Maturity:** Audit all "Vision" statements to reflect real-world 2026 achievements and team growth.
