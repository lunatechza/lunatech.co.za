Welcome to the official repository for the Lunatech (Pty) Ltd. website. Founded in 2014, Lunatech has evolved into a leading South African software development powerhouse, now fully focused on the 2026 landscape of AI-driven innovation and intelligent automation.

## 🚀 Vision 2026: AI-First

As we navigate the digital revolution, Lunatech has pivoted to an **AI-First** development philosophy. We leverage autonomous agents, Large Language Models (LLMs), and decentralized technologies to build the next generation of business solutions.

### Core Pillars
- **AI & Automation:** Custom agentic workflows and LLM fine-tuning.
- **Next-Gen Development:** AI-augmented software engineering for rapid delivery.
- **Blockchain Integration:** Intelligent digital asset management (e.g., ChainEX).

## 🛠 Tech Stack

Our 2026 technology stack is built for performance and intelligence:
- **Languages:** Python (AI/ML), Rust, Go, TypeScript.
- **AI Frameworks:** LangChain, LlamaIndex, PyTorch, TensorFlow.
- **Frontend:** Bootstrap 3 (Standardized), jQuery, FontAwesome 4.7.
